This R Shiny app is an interactive tool for quickly reviewing linear mixed models, letting users pick a response variable, up to three fixed effects, random effects, and a transformation (log, sqrt, etc.) from drop-down menus to fit a model with `lmerTest::lmer`. It produces a Kenward-Roger ANOVA table, residual and random-effects diagnostics, estimated marginal means with Tukey-adjusted post-hoc comparisons and compact-letter-display groupings, plus raw-data exploration plots, model fit statistics, influence diagnostics, and an optional model comparison. Built-in guards validate the spec before fitting (blocking empty or oversized effect lists, invalid transformations, and single-level factors) so the underlying engine fails gracefully with plain-language warnings rather than crashing.
